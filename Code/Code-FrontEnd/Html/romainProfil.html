C'est qu'on se sent bien d'être en vacances. On a l'impression qu'on a le temps de tout faire. Je sais que ça n'est pas le cas, comme tous les autres, le temps me mange. Quelques jours à me morfondre, à travailler, à fumer, puis déjà, je retrouverais ce petit ange bleu...Pourquoi bleu ? Je ne sais pas, peut-être pour Ses yeux. J'ai horreur du bleu habituellement, préférant le vert, ce vert que je vois dans les champs, dans les pots, des mes vêtements. Ce vert que je mange, que je défais de sa terre, ce verre dans lequel je bois un peu d'alcool.
